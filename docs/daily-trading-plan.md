# A 股每日交易计划方案

这份方案用于把当前“每日股票分析”从宽泛日报，改成更贴近交易节奏的三段式推送：盘前定框架，盘中看验证，盘后做复盘。目标不是让 AI 给出直接买卖指令，而是稳定输出观察条件、确认信号、失效信号和风险等级。

## 总体原则

- 少讲背景，多讲当天能验证的盘面事实。
- 先看市场，再看主线，再看个股。
- 每条结论都必须有数据或新闻触发因素。
- 不输出“必涨、必跌、满仓、梭哈”等绝对化建议。
- 对股票只给观察计划：关注、等待确认、降低关注、风险升高。
- 飞书推送优先短而清楚，详细内容放在折叠式结构里。

## 每日节奏

| 时间 | 名称 | 目标 | 建议运行模式 |
| --- | --- | --- | --- |
| 08:40 | 盘前计划 | 明确今日市场环境、潜在主线、风险事件和自选股观察位 | 新增 workflow |
| 10:30 | 早盘观察 | 判断开盘后资金方向、热点强度、盘前主线是否被验证 | 新增 workflow，可选 |
| 14:00 | 午后确认 | 判断主线延续还是轮动退潮，避免尾盘被动 | 新增 workflow，可选 |
| 17:30 | 盘后复盘 | 总结今日市场性质、主线成败、明日观察计划 | 由现有 18:00 日报改造或新增 workflow |

## 盘前计划模板

标题：`A股盘前计划 YYYY-MM-DD`

输出结构：

1. 今日市场环境
   - 外围市场：美股、港股、人民币、商品、重要事件。
   - 国内消息：政策、监管、产业、财报、公告。
   - 风险事件：重要会议、经济数据、限售解禁、业绩披露。

2. 今日大盘观察框架
   - 指数状态：上证、深成指、创业板、科创方向。
   - 情绪状态：昨日涨跌家数、涨停跌停、连板高度、成交额。
   - 今日关键问题：放量还是缩量，修复还是退潮，权重还是题材。

3. 今日主线候选
   - 只列 2-4 条。
   - 每条包含：触发因素、观察板块、核心股票、验证条件、失效条件。

4. 自选股观察计划
   - 每只股票只输出三点：
     - 今日角色：强势跟踪 / 中性观察 / 风险升高。
     - 确认信号：放量突破、板块共振、资金回流等。
     - 失效信号：跌破关键位、缩量冲高、资金流出等。

5. 今日纪律
   - 不追什么。
   - 等什么信号。
   - 风险升高时怎么降级观察。

## 盘中观察模板

标题：`A股盘中观察 HH:mm`

输出结构：

1. 盘面温度
   - 指数涨跌、成交额变化、上涨/下跌家数。
   - 涨停/跌停数量，连板高度，炸板情况。
   - 市场状态：进攻 / 轮动 / 防守 / 退潮。

2. 资金动向
   - 主力资金方向。
   - 行业/概念资金净流入前列。
   - 资金流出明显板块。
   - 是否出现权重护盘或题材抱团。

3. 热点验证
   - 盘前候选主线：验证 / 未验证 / 分歧。
   - 新出现热点：触发因素、强度、持续性判断。
   - 热门股票：只列能代表板块强度的核心票，不做追涨建议。

4. 自选股状态
   - 是否强于所属板块。
   - 是否满足盘前确认信号。
   - 是否触发失效信号。

5. 午后/尾盘观察
   - 只给 3 条以内。

## 盘后复盘模板

标题：`A股盘后复盘 YYYY-MM-DD`

输出结构：

1. 今日市场定性
   - 强修复 / 弱修复 / 震荡 / 退潮 / 防守。
   - 用指数、成交额、涨跌家数、涨停跌停、板块持续性支撑结论。

2. 今日真正主线
   - 最强 1-3 条主线。
   - 每条说明：为什么强，核心股票是谁，是否有持续催化。

3. 资金与情绪
   - 热门板块资金流入。
   - 明显流出板块。
   - 情绪指标：涨停、跌停、连板、高位股反馈。

4. 自选股复盘
   - 今日表现是否强于大盘和所属板块。
   - 今天信号：确认 / 未确认 / 失效。
   - 明日观察条件。

5. 明日计划
   - 优先观察方向。
   - 不碰方向。
   - 大盘关键确认信号。
   - 风险提示。

## 数据字段清单

当前项目已能覆盖：

- 指数行情：上证、深成指、创业板等。
- 涨跌家数、涨停跌停数量。
- 成交额。
- 板块涨跌排行。
- 自选股行情、技术面、资金/量价信息。
- 新闻资讯：通过博查搜索。
- AI 总结：通过 DeepSeek。
- 飞书推送。

需要进一步确认或增强：

- 热门股票排行榜。
- 连板高度、炸板率、涨停原因。
- 更细的行业/概念资金流向。
- 北向资金。如果接口不可用，则不强依赖。

## 飞书推送格式

盘前推送建议控制在 800-1500 字。

```text
【A股盘前计划】2026-05-16

市场环境：中性偏谨慎
今日关键词：成交额、科技主线延续、资金是否回流

一、今日看盘框架
1. 指数：关注创业板/科创是否强于上证
2. 情绪：看涨停数量和跌停数量是否改善
3. 量能：成交额不足时不追高

二、主线候选
1. 半导体：验证信号是核心票放量走强
2. 电力：验证信号是板块内多只个股同步走强

三、自选股观察
688126：中性观察，等待板块共振
688111：风险升高，若继续弱于板块则降低关注

四、今日纪律
不追缩量冲高；只看放量确认和板块共振。
```

## 实施路线

第一步：先保留现有每日 18:00 推送，新增一份“交易计划模板”和变量设计。

第二步：新增 `TRADING_PLAN_MODE`，支持：

- `pre-market`
- `midday`
- `afternoon`
- `post-market`

第三步：新增 GitHub Actions：

- `08:40 pre-market`
- `10:30 midday`
- `14:00 afternoon`
- `17:30 post-market`

第四步：先只启用盘前和盘后，观察 3-5 个交易日后再开启盘中推送。

## 建议先启用的版本

为了避免飞书消息太多，第一版建议只做两条：

- 08:40 盘前计划。
- 17:30 盘后复盘。

10:30 和 14:00 作为第二阶段再开。
